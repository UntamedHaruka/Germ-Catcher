🎮 Play the Game

## 🧠 Description

Germ Catcher is a 2D Unity game where the player interacts with moving elements to catch germs and score points. The game demonstrates core Unity concepts like player movement, object spawning, and collision detection.

## 🛠 Tech Stack

* Unity (2D)
* C#
* Visual Studio

## 🎯 Features

* Player movement
* Enemy / object spawning system
* Collision detection
* Score system

## ▶️ How to Run

1. Download the project (endless.zip)
2. Extract it
3. Open in Unity Hub
4. Click Play

## 📁 Project Structure

* Assets → Scripts, Scenes, Prefabs
* Packages → Dependencies
* ProjectSettings → Unity configuration
